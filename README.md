# Macaw Mod

Adds a passive **Macaw** bird to Minecraft 1.20.1 (Fabric):

- Spawns naturally in **jungle biomes**.
- Tame it by feeding it **seeds** (wheat, melon, pumpkin, or beetroot seeds).
- Once tamed, right-click it to make it **perch on your shoulder** (just like a vanilla parrot).
- While a tamed Macaw follows you or rides your shoulder, you gain **Night Vision**.
- Right-click again to toggle sitting.

## Build the mod for FREE using GitHub (no Java install needed)

You do **not** need Java or any programming tools on your computer. GitHub will build the `.jar` for you for free.

### Steps

1. **Create a GitHub account** at https://github.com (free).
2. Click the **+** in the top-right → **New repository**. Give it any name → **Create repository**.
3. On the new empty repo page, click the **"uploading an existing file"** link.
4. **Extract the downloaded zip** of this mod on your computer.
5. **On macOS, press `Cmd + Shift + .` (period) in Finder** to show hidden files.
   - ⚠️ **Important:** the `.github` folder is **invisible by default on macOS**. You MUST press `Cmd + Shift + .` before selecting files, or the build workflow will not be uploaded and the build will **never run**.
6. Open the extracted folder, then **select ALL files and folders from INSIDE it** — including the hidden `.github` folder. **Do not drag the outer folder itself — drag its contents.**
7. Drag everything into the GitHub upload page.
8. Scroll down and click **Commit changes**.
9. Click the **Actions** tab at the top of your repo.
10. Wait about **2 minutes** for the build to finish (green check mark).
11. Click into the completed run and download the **mod-jar** artifact.
12. Unzip it and copy the `.jar` file into your `.minecraft/mods/` folder.

Make sure you also have [Fabric Loader](https://fabricmc.net/use/installer/) and the [Fabric API](https://modrinth.com/mod/fabric-api) installed for Minecraft 1.20.1.

Enjoy your Macaw! 🦜
